# 关于项目
这是一个基于Ursina制作的优化库插件
本项目采用中文作为主要展示语言方便中国区开发者更好的开发和使用

# 开源与分发
## 协议
本项目采用Ling自定义开源协议，开发者可使用其进行游戏开发但不允许对源代码进行修改，或改名后再次分发

## 授权
这个项目授权给所有使用Ursina引擎的开发者，开发者可在遵守协议的前提下使用本项目的优化功能

## AI人工智能辅助开发协议
### 规范
在使用Ai IDE 如（VS Code，TraeCN）时，开发者需要明确告知AI智能体不得在代码中添加注释
在使用AI IDE时开发者需要明确告知AI智能体不得修改 \更好的Ursina 目录下 除 cfg 文件外的其他文件

# 其他协议
- 当开发者使用本插件造成法律指控，原作者概不负责，原作者的职责仅限于更新与维护

# 使用要求
## python
版本不得低于 3.12.x 否则原作者无法保证代码中的汉字变量名等产生的兼容性问题

## 依赖
使用时必须有Ursina引擎存在否则无法运行，因为它只是为Ursina引擎提供优化和光照功能的插件级优化
